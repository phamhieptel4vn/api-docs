HTTP Request​

GET https://{{API_HOST}}/v3/report/call/user

Query Parameters​

ParameterDescriptionExampleRequired
start_timeThống kê theo khoảng thời gian2023-03-07%2000%3A00%3A00
end_timeThống kê theo khoảng thời gian2023-03-07%2023%3A59%3A59

curl --location 'https://{{API_HOST}}/v2/report/extension' \
